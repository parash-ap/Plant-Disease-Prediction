## 📌 Project Overview

Plant Disease Prediction is a Deep Learning-based image classification system that detects plant diseases from leaf images.

The model is trained using Convolutional Neural Networks (CNN) to classify healthy and diseased plant leaves accurately.

This project helps in early disease detection and supports smart agriculture solutions.

---

## 🚀 Features

- 🌿 Leaf Image Classification
- 🧠 CNN-based Deep Learning Model
- 📊 High Accuracy Model
- 🖥 Simple and User-Friendly Interface
- ⚡ Fast Prediction System

## ⚙️ Installation

1. Clone the repository

2. Install dependencies
pip install -r requirements.txt

3. Run the project
python manage.py runserver
